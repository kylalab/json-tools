Quick JSON 소개

Quick JSON은 브라우저에서 바로 사용하는 JSON 유틸리티 모음입니다. Pretty Print(가독성 있게 정렬), Minify(공백 제거), Validate(유효성 검사), JSON ↔ CSV 변환을 지원합니다. 모든 처리는 서버 전송 없이 브라우저 내에서만 수행되므로 안전하게 사용할 수 있습니다.

  • 빠른 포맷팅/압축, 오류 위치 안내 유효성 검사
  • JSON → CSV / CSV → JSON 양방향 변환, 구분자(콤마/탭/세미콜론/파이프) 선택
  • 결과 Copy / Download 버튼 제공, 다크 모드 지원

사용 방법

1) 포맷팅(Pretty) / 압축(Minify)

  1. 왼쪽 입력창에 JSON을 붙여 넣습니다.
  2. Pretty Print 또는 Minify 버튼을 누릅니다.
  3. 오른쪽 결과 영역에서 Copy Output 또는 Download로 저장합니다.

2) 유효성 검사(Validate)

  1. 입력창에 JSON을 붙여 넣고 Validate 버튼을 누릅니다.
  2. 정상일 경우 “Valid JSON” 메시지, 오류 시 위치/원인을 안내합니다.

3) JSON → CSV

  1. 배열 형태의 JSON(예: [{"id":1,"name":"alice"},{"id":2,"name":"bob"}])을 입력합니다.
  2. 필요하면 Delimiter에서 구분자(콤마/탭 등)를 선택한 뒤 JSON → CSV를 누릅니다.

4) CSV → JSON

  1. 첫 줄에 헤더가 있는 CSV(예: id,name\n1,alice\n2,bob)를 입력합니다.
  2. CSV의 구분자에 맞춰 Delimiter를 선택한 뒤 CSV → JSON을 누릅니다.

예시

JSON 입력 예시

{
  "team": "alpha",
  "members": [
    {"id": 1, "name": "alice"},
    {"id": 2, "name": "bob"}
  ]
}

CSV 입력 예시

id,name
1,alice
2,bob

자주 묻는 질문(FAQ)

Q1. 내 데이터가 서버로 전송되나요?

아니요. 모든 기능은 브라우저에서만 동작하며, 입력/출력 데이터는 서버로 전송되지 않습니다.

Q2. 지원하는 구분자는 무엇인가요?

콤마(,), 탭(\t), 세미콜론(;), 파이프(|)를 지원합니다. 드롭다운에서 선택하세요.

Q3. 어떤 JSON 구조가 CSV로 변환되나요?

동일 키를 가진 객체의 배열 형태([{...},{...}])를 권장합니다. 중첩된 구조는 평탄화되지 않으며, 중첩이 있을 경우 문자열로 직렬화될 수 있습니다.

Q4. 오류가 나면 어떻게 하나요?

Validate로 유효성 확인 후, 오류 메시지의 위치를 참고해 쉼표, 따옴표, 대괄호/중괄호 균형을 점검하세요.

Q5. 다크 모드/단축키가 있나요?

상단의 Dark Mode 토글을 사용할 수 있습니다. (향후 단축키 지원 예정)

Q6. 개인정보/보안은 어떻게 처리하나요?

입력한 JSON/CSV는 저장되지 않으며, 브라우저 메모리에서만 처리됩니다. 광고 식별자는 제3자(구글) 정책을 따릅니다.
