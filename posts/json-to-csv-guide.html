JSON ↔ CSV 변환 완전 가이드(실무 예시 포함)

2025-09-28 · 작성자 Kyla

분석팀은 CSV를, 개발팀은 JSON을 선호합니다. 이 글에서는 배열 형태의 JSON을 CSV로 바꾸는 안전한 규칙, 구분자 선택(콤마/탭/세미콜론/파이프), 헤더 처리, 중첩 구조 대응을 정리합니다.

권장 JSON 형태

[
  {"id":1, "name":"alice"},
  {"id":2, "name":"bob"}
]

CSV 출력 예시

id,name
1,alice
2,bob

실무 팁

  • 중첩 객체는 문자열화하거나 별도 컬럼으로 평탄화(룰을 문서화)
  • 긴 숫자 ID는 문자열로 취급(정밀도 보존)
  • BOM/인코딩(UTF-8)과 줄바꿈(\n) 일관성 유지

변환은 Quick JSON 도구에서 브라우저 내 처리로 안전하게 수행됩니다. 결과는 Copy/Download로 바로 저장할 수 있습니다.
